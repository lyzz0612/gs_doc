# 简介

GS语言，全称Game Script，是专门为游戏开发这一垂直领域设计的一个动态语言，源自我们之前的自研语言LPC2.0，再往前追溯源自MudOS

:::tip

🎥 点击查看 [**GS语言的前世今生**](https://leiting.feishu.cn/wiki/OLOJwce1Ui6iRQkPcMWcjatRnbg)

:::
